Integrate

Innovate

To innovate this code, I outline the letters with dots that would then start moving with perlin noise

then made each dot have a random color

made it so when you clicked the mouse it changed the direction of the noise

i then removed the background from the draw function so that you can see the black lines when the dots moved

i then made it so the dots rotate together
